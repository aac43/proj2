SOFTWARE

Look back at the first printer after the dot matrix printer! The original LaserWriter I was released in 1985 by Apple Computers.

ADOBE POSTSCRIPT

Software suitable for being a page description language by Adobe created to use with laserwriters and typesetters. Currently Adobe still sells Postscript 3, which is the third version of the language. Postscript was the secret to making the laser printer print exactly what is seen on screen.

ADOBE PAGEMAKER

Software introduced in 1985 by Aldus and is known as one of the first desktop publishing programs. Aldus was then bought by Adobe and the projects were continued as 'Aldus Pagemaker 1.0, 1.2, 2, 3, 4, 6.0, 7.0. A screenshot of the latest version of this historic software has been provided below.
